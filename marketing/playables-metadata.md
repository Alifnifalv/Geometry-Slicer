# Geometry Slicer Playables Metadata

## Short Description

Slice geometric shapes into perfectly equal pieces using a limited number of cuts.

## Long Description

Geometry Slicer is a touch-friendly precision puzzle game where players slice polygons, circles, stars, and complex symmetric shapes into equal-area pieces. Each level gives a target piece count and a limited number of valid cuts, rewarding spatial reasoning, clean center cuts, and replayable accuracy mastery.

## Feature Bullets

- 100 progressively harder levels across 10 chapters.
- First-run tutorial with animated cutting guidance.
- Level map with unlocked progression and best-grade tracking.
- Precision scoring with Perfect, Great, Close, and Cleared grades.
- Failure feedback that explains whether the player missed piece count or area balance.
- Lightweight synthesized audio, pause/resume handling, and Playables cloud-save support.
